Introduction

Si tu lis ce guide, c’est probablement que tu est une personne à qui l’accès a été donné.

Ce logiciel permet d’héberger et de partager des fichiers via une base de données. Merci de ne pas redistribuer ce logiciel sans autorisation.

Mega Files

Le panneau "Mega Files" affiche tous les fichiers actuellement hébergés sur la base de données.

  • Nom du fichier
  • Description du fichier
  • Chaque fichier télécharger sera sur le bureau dans un dossier download(Possible que sa soit modifier dans de future maj)

Bouton Refresh

Le bouton Refresh permet d’actualiser la liste des fichiers.

  • Si un nouveau fichier est ajouté
  • Si un fichier est supprimé
  • Si la liste n’est plus à jour

Post (Envoyer un fichier)

Le bouton Post permet d’envoyer un fichier vers la base de données.

Règles importantes :

  • Taille maximale : 500 Gb
  • Les fichiers ZIP comptent comme un fichier normal
  • Une description peut être ajoutée
  • La description apparaît à droite du nom du fichier
  • L'application se rafraîchit automatiquement après l'envoi
